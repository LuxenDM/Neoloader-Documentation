lib.uninstall
Definition:
  • lib.uninstall (verification_key) :: nil
Description:
  • Attempts to remove Neoloader from the user's device
Arguments:
  1. string verification_key: Authentication key given to the active manager
Returns:
  • nil
Other notes:
  • This function should only be called by the user directly (using the /neodelete command) or by Neoloader's active manager.
