lib.reload
Definition:
  • lib.reload () :: Game Reloads
Description:
  • Forces an interface reload
Arguments:
  1. none
Returns:
  • none
Other notes:
  • Neoloader will attempt to clean up all user commands not present in the "zcom.lua" file. Whether this is still neccesary is unknown, but for a time the game wasn't cleaning up commands properly, resulting in 'ghost commands' attempting to access data that had been garbage collected by the lua engine. This functionality remains in Neoloader to help make sure plugins are unloaded as much as possible, just in case.
  • Neoloader registers a user command "reload" which points to this function.
