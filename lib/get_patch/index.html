lib.get_patch
Definition:
  • lib.get_patch () :: Neoloader_patch_level
Description:
  • Retrieves the current patch level of Neoloader
Arguments:
  • None
Returns:
  • number patch_level: A number describing the patch level of the API
Other notes:
  • Every *addition* to the API increases the patch level. The API value will only change if there is an incompatibility with previous functions.
