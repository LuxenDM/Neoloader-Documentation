lib.notify
Definition:
  • lib.notify (notification_status, ...) :: nil
Description:
  • Sends a notification to Neoloader's active manager
Arguments:
  1. string notification_status: the status to be communicated
  2. ...: any additional values relevant to the communication
Returns:
  • nil
Other notes:
  • The behavior of the notification system is dependent on the activated manager; for the bundled neomgr, only a few statuses are expected, and the rest are ignored.
  • This feature is meant to be used for communicating to the user through a unified method; for internal use, Vendetta Online's existing event management system works perfectly fine.
