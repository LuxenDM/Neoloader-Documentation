lib.activate_mod
Definition:
  • lib.mod_read_str (id, version, header, key) :: ini_value
Description:
  • Reads the key item from the selected mod's registration INI file
Arguments:
  1. string id: name of the mod to be read from
  2. string version: version of the mod to be read from
  3. string header: The [header] to look under
  4. string key: the key of the item to check
Returns:
  • string ini_value: The value stored in the INI file at [header], key. "" is returned if the key doesnt exist.
Other notes:
  • If repeated access is required, obtaining the mod's registration path and using gkini.readstring2 may be more efficient.
