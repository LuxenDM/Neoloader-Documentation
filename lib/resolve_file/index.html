lib.resolve_file
Definition:
  • lib.resolve_file (file, path, path_2) :: nil
Description:
  • Attempts to load and execute a file using a variety of path options. This function attempts to make file loading behavior consistant across the game's state.
Arguments:
  1. string file: file name to be accessed
  2. string path: path to the file to be accessed
  3. string path_2: alternate path to check
Returns:
  • boolean status: true if the file loaded successfully
  • ...: results of the file, or a string error message
Other notes:
  • if path is nil, this function attempts to split the file argument into a path/file pair for you.
