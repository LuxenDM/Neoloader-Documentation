lib.get_latest
Definition:
  • lib.get_latest (id) :: mod_version
Description:
  • Retrieves the latest version of an installed mod
Arguments:
  1. string id: name of the mod to look up
Returns:
  • string mod_version: the latest version of the installed mod
Other notes:
  • Other API functions, when given mod version "0", use this function to handle lookup
  • This function gets the latest INSTALLED, not the latest version that is activated.
